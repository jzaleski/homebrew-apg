# Homebrew - jzaleski

Tap of `[home]brew` formulae maintained by @jzaleski

## Requirements

* [Homebrew](https://github.com/Homebrew/brew)

## Installation

Run the following on the command-line:

```sh
$ brew tap jzaleski/homebrew-jzaleski
```
